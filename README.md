## Краткое описание программы

**Notebook Organizer** — это консольное приложение для управления заметками, разработанное на Python. Программа позволяет создавать, редактировать, удалять и фильтровать заметки различных типов с возможностью отмены действий и автоматическим сохранением в JSON-файл.

### Основные возможности

- **Три типа заметок (полиморфизм):**
  - 📝 Текстовая заметка
  - 🎤 Голосовая заметка (с указанием длительности)
  - ✅ Чек-лист (со списком задач)

- **Полный CRUD-функционал:**
  - Create (создание)
  - Read (просмотр)
  - Update (редактирование)
  - Delete (удаление)

- **Система отмены действий (Undo/Redo):**
  - Реализована с использованием структуры данных Стек (Stack)
  - Позволяет отменять и повторять операции

- **Фильтрация заметок:**
  - По тегам
  - По дате (конкретная дата или диапазон)
  - По поисковому запросу (по заголовку и тексту)

- **Автосохранение:**
  - Все заметки автоматически сохраняются в JSON-файл
  - При запуске данные загружаются из файла

- **Валидация ввода:**
  - Проверка заголовка на пустоту и длину (не более 100 символов)
  - Проверка текста на пустоту

## Как запустить приложение

### Системные требования

- Python 3.7 или выше
- Любая операционная система (Windows, macOS, Linux)
